# ⚾️ 야구게임 (BULLS AND COWS)

💁‍♀️ 랜덤으로 주어지는 4자리의 숫자를 맞추는 야구게임
<br><br>

## ⚙️ Install
1. clone(SSH) : git@github.com:YouJin-Cho/BULLS-AND-COWS.git
2. terminal ➡️ cd BULLS-AND-COWS
3. npm install
4. npm run start
<br><br>

## 💪🏻 기술 스택
* JavaScript
* React
<br><br>


## 👀 우와 야구게임이다!
